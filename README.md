# Image-to-PDF Converter

## Description
A simple Python GUI application that converts multiple images into a single PDF file.  
Users can select multiple PNG/JPG/JPEG images, view the selected images, and save them as a PDF with a custom name.

## Features
- Select multiple images at once
- Preview selected images
- Enter custom output PDF file name
- Converts images to a well-formatted PDF

## Installation
1. Clone the repository:
```bash
git clone https://github.com/Dhriti2220/Image-to-PDF-converter.git
cd Image-to-PDF-converter

2. Install dependencies:
pip install -r requirements.txt

Tkinter is usually included with Python. Make sure Python >=3.8 is installed.

Usage
Run the application:
python image_to_pdf_converter.py
